Docker monitoring
-----------------
1. docker stats
2. curl --unix-socket /var/run/docker.sock  http://localhost/containers/<container-name | cid>/stats 
e.g.
	docker run -it alpine:latest /bin/sh
	curl --unix-socket /var/run/docker.sock  http://localhost/containers/<id>/stats
